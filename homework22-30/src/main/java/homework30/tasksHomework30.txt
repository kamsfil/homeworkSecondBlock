Основное задание
1. Создать Maven проект используя archetype:generate интерактивный режим.
Во время выполнения ознакомиться со списком возможных архетипов.
Выбрать архетип, версию архетипа, указать groupId, artifactId, версию.
Открыть проект и выполнить maven команду install и проанализировать
результат работы команды (убедиться, что появился артефакт в локальном репозитории).

Дополнительное задание
2. Создать Ant проект и выполнить сборку проекта (опционально).
3.* Создать свой собственный Maven архетип (https://habr.com/ru/post/564646/).





Шаги для создания базового архетипа из текущего проекта:
1. запускаем архетип для создания архетипов из текущего проекта
mvn archetype:create-from-project
скачиваются зависимости и появится папка в target/generated-sources с базовыми настройками архетипа

2. переходим в папку
cd target/generated-sources/archetype
можете здесь посмотреть структуру файлов, поменять на усмотрение

3. устанавливаем архетип в локальный репозиторий
mvn clean install
после можно зайти в .m2/repository и при успехе будет создан файл archetype-catalog.xml

4.0 можете создать заранее пустую директорию под новый проект
4. запускаем интерактивный вид для генерации проекта из архетипа
mvn archetype:generate
последний должен быть типа local с вашим архетипом
дальше уже как обычно устанавливаем проект
