Основное задание
1. Создать шаблон проекта используя систему сборки проекта Gradle.
Собрать jar файл, проанализировать результат.
2. Создать программу для сохранения заказа (Order) на покупку товара (Item).
Сделать логирование действий пользователя в файл.
Предусмотреть различные уровни логирования: общая информация,
предупреждающие сообщения, сообщения об ошибках, дебаг
сообщения.
2.1 Для предыдущего задания создать unit-тесты. Предусмотреть покрытие
кода тестами минимум 50%.

Дополнительное задание
3. Представим, что Java не имеет арифметической операции деления.
Создать программу для реализации метода деления одного целого
числа на другое. Создать unit-тесты для программы.
4. Добавить в Gradle дополнительные задачи (https://coderlessons.com/tutorials/raznoe/vyuchit-gradle/gradle-zadachi):
   - типа Copy для копирования всех файлов из build/resources в папку files
     (предварительно создайте тестовые файлы), добавить зависимость на задачу compileJava,
     выполните задачу и убедитесь, что файлы скопировались в папку files
   - типа Delete для удаления содержимого папки files
