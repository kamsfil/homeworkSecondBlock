Основное задание
1. Написать простое приложение (должно быть от 2 сущностей и соответствующих таблиц в БД),
в котором сущности будут связаны (на уровне SQL через JOIN) и продемонстрировать в логах
CRUD (create, read, update, delete) операции с БД.

Дополнительное задание
2. Перенести приложение на структуру entity, repository, service из SOLID темы, где repository
отвечает за работу с БД.
3. Добавить поддержку Servlet (сформировать web приложение и перенести структуру папок из пункта 1
+ папка controller, где будут храниться сервлеты), добавить jsp страницы для CRUD операций.
4. Добавить логирование и покрытие тестами более 60%.
